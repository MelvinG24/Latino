# Desinstalar Latino
Para desinstalar Latino procedemos con los siguientes pasos:
* **Inicio >**
    - **Panel De Control >**
        - **Desinstalar un Programa ...**
        
(llegado a este punto, solo tenemos que buscar el programa de Latino y darle a desinstalar)
